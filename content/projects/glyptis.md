# Glyptis

## App Store
- https://apps.apple.com/us/app/glyptis-realidade-esculpida/id6755839447

## What problem did this solve?
Glyptis makes digital sculpture feel physical by allowing users to create voxel models and place them in AR environments.

## What was my role?
- Role title: iOS/AR Developer
- Team size: 5
- Scope owned: 3D canvas architecture, AR flow, persistence layer, SwiftUI bridge with 3D frameworks

## What I specifically built
- Implemented unified coordination between SwiftUI UI and 3D/AR runtime flows
- Built voxel modeling pipeline with persistent local structures
- Optimized scene updates for smoother interaction in creation and AR placement modes
- Structured model relationships for reliable offline persistence

## Business impact
- Architecture: Cleaner integration between UI layer and 3D/AR runtime
- Performance: Stable interaction loop during modeling and AR placement
- Data reliability: Safer persistence for complex sculpture sessions

## How I delivered
- Key decisions: Voxel-based editing for predictable performance and user learning curve
- Trade-offs: Prioritized robust offline persistence before cloud sync expansion
- Delivery model: Collaborative design-development iteration for usability in spatial UI
