# Shiro

## App Store 
- https://apps.apple.com/us/app/shiro/id6752502968

## Qual problema isso resolveu?
Shiro recria a sensação de jogos arcade portáteis clássicos com performance moderna no iOS e integrações nativas.

## Qual foi meu papel?
- Cargo: Desenvolvedor iOS/Game
- Tamanho do time: 5
- Escopo: Sistemas de gameplay com SpriteKit, arquitetura, integração com Game Center e overlays em SwiftUI

## O que eu construí especificamente
- Fluxos de máquina de estado com GameplayKit para comportamento do jogador
- Otimizações de cenas SpriteKit para manter frame pacing estável
- Integração de leaderboards e autenticação com GameKit
- Sistemas de feedback tátil/sonoro para melhorar game feel

## Impacto de negócio
- Engajamento: loop competitivo com leaderboard nativa
- Performance: renderização estável em cenas de alta ação
- Qualidade: estrutura modular para facilitar evolução

## Como eu entreguei
- Decisões-chave: combinação de runtime SpriteKit com shell/HUD em SwiftUI
- Trade-offs: stack nativa Apple em vez de engine cross-platform para integração mais fina
- Modelo de entrega: arquitetura modular entre estados, nodes e camada de UI
