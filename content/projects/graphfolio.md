# Graph Folio

The portfolio you are looking at. Built with [[Astro]], [[D3.js]], and [[TypeScript]].

Every node is a markdown file. Every edge is a [[wikilink]].

Inspired by the [[Unix Philosophy]] — plain text as the universal interface.
