# Unix Philosophy

Write programs that do one thing and do it well.

Write programs that work together.

Write programs that handle text streams.

Applied to [[Graph Folio]]: plain text markdown as source of truth, each component with a single responsibility.

## Core tenets

1. **Do one thing well** — each module has one purpose
2. **Text streams are universal** — text connects everything
3. **Build early prototypes** — iterate fast
4. **Use tools in preference to unskilled help** — automate
