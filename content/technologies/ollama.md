# Ollama

## Why I use it
Ollama enables local AI inference with full code privacy and predictable latency, which is ideal for developer tooling.

## Proven outcomes
- 100% local inference for privacy-sensitive workflows
- Lower latency compared with remote API round trips
- Quick model switching for prompt and quality experiments

## Level of proficiency
- Intermediate
